Se juega a través de un dispositivo multimedia (consola, computador, etc.)

En el caso de los videojuegos, cuando se habla de un dispositivo audiovisual, esto se refiere al aparato mediante el cual se puede correr y operar un videojuego. En palabras simples: el dispositivo audiovisual es una plataforma multimedia, como la consola de juego o el computador utilizado para jugar.

Este aparato se caracteriza por enviar, recibir y procesar datos entre el interior y el exterior del sistema de forma audiovisual. Por ejemplo, al correr un videojuego, en la consola se procesan los datos grabados en el cartucho, disco o archivo. Luego, esta interpreta esa información y muestra las imágenes, animaciones y otros elementos audiovisuales que conforman el juego. Finalmente, el jugador interactúa con esta información, tecleando o presionando botones con el control, para avanzar en el juego. Todo este proceso se repite constantemente durante la duración de esta actividad.
